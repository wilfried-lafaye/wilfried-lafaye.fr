NYC Accidents SQL Model

A data engineering and analysis project focused on transforming raw NYC traffic accident data into a structured relational database. It features interactive dashboards and a SQL interface to explore the data.

Application requires local setup.

View on GitHub

* This is a Streamlit application designed to run locally.

Technical Details

  • Data Pipeline: SQLite3, Pandas, Normalization (3NF)
  • Backend: Python 3.10+
  • Frontend: Streamlit
  • Visualization: Plotly Express
  • Source Code: GitHub Repository
